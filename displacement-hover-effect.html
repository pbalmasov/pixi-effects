Установка

npm install git+http://gitlab.peppers-studio.ru:88/peppers/190516_1.git

Импортирование эффекта

import 'flip-effect/'

Создание элемента

Используя data атрибута принимающего json объекта в виде строки: 

<div data-displacement-effect='{"options":{"frontImage":"assets/displacement/images/chebur.jpg","backImage":"assets/displacement/images/karlson.jpg","offset":1}}'></div>

Свойство appOptions объекта передает параметры в конструктор PIXI.Application

Свойство options объекта:

frontImage (String)
- ссылка на переднюю текстуру

backImage (String)
- ссылка на заднюю текстуру

offset (Number)
- запаздывание(на сколько сильно закручивается) (0 - не закручивается, 0.2 - по умолчанию, 1 - сильно закручивается)

duration (Number)
- время переворота

isFlip (Boolean)
- первая показываемая сторона

Опции:{
  options: {
    intensity1: 0.1,
    intensity2: 0.1,
    angle2: 180,
    image1: 'assets/displacement/images/ny.jpg',
    image2: 'assets/displacement/images/ny-info.jpg',
    displacementImage: 'assets/displacement/stripe1mul.png'
  }
}
